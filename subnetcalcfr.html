Calculateur de Sous-Réseau en Java Interface Utilisateur Graphique en Java

Le projet de réseau consistait à créer un calculateur de sous-réseau dans notre langue préférée. À l'origine, je l'ai créé en tant que programme basé sur une console, mais j'ai récemment créé une interface graphique rapide pour m'entraîner davantage et me préparer pour ma deuxième année d'ingénierie informatique.

Cliquez sur la vidéo pour la lire.

Ce court clip montre comment fonctionne le programme. Il prend une adresse IP et un masque de sous-réseau, renvoie des informations sur ce sous-réseau exact et vous permet d'afficher tous les sous-réseaux possibles dans un format de tableau.
